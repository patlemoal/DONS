Fichiers htlm

"base"
code pour la navigation des pages, cela permet de garder la même trame pour chaque page et d'aller d'une page à l'autre

"page" (ou acceuil sur la navigation)
page qui présente l'association

"formulaire"(ou "Nous soutenir" sur la navigation)
page qui permet au donnateur de compléter ses coordonées, le montant de ses dons et ses coordonnées bancaires

"afficher les dons" (ou administrateur sur la navigation)
page qui permet de récupérer le montant des dons par personnes, l'adresse mail et le total des dons.

Fichier app.py

- connection à la base de données sur Mongo Atlas 
- code pour le routage url
- fonctions qui permettent de transposer les données vers/de Mongo Atlas au fichier python
